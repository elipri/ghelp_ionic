To add a new task or a new plant, simply click on the plus sign.

Tasks do not require a description, but you can add it if you wish to do so. Clicking on a task will show you the description (if there is one).

Any new plant has to have a name and a place where it resides. You can pick a place from a list or type in a new one. You can also add a photo and a description (but these are not required).
